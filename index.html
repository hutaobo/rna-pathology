RNA-Pathology

A schema-driven database framework for spatial molecular pathology.

Database Schema Roadmap GitHub

Overview

RNA-Pathology is an open research project aiming to organize and standardize spatially resolved molecular pathology data, with a particular focus on RNA-based spatial technologies.

The project emphasizes schema consistency, spatial context, and reproducibility, serving as a foundation for computational pathology, spatial omics method development, and cross-study comparison.

Why RNA-Pathology?

Spatial molecular datasets are rapidly increasing in complexity and volume, yet remain fragmented across formats, technologies, and analytical assumptions.

RNA-Pathology addresses this by adopting a schema-first approach, where spatial structure, biological annotations, and molecular measurements are treated as first-class entities rather than auxiliary metadata.

What This Project Is (and Is Not)

RNA-Pathology is:

  • A conceptual and technical framework for spatial pathology data
  • A curated, schema-driven database architecture
  • A foundation for reproducible spatial analysis

RNA-Pathology is not:

  • A general-purpose data dump
  • A visualization-first portal
  • A closed or proprietary platform

Current Status

RNA-Pathology is currently in an early public architecture phase.

  • Schema definition: active
  • Documentation: available
  • Data ingestion: limited pilot datasets
  • Programmatic access: planned
