Database Overview

RNA-Pathology is a schema-driven framework for organizing spatial molecular pathology data, connecting tissue context, spatial coordinates, and reproducible computational analysis.

What is RNA-Pathology?

RNA-Pathology is an evolving open research database designed to organize, standardize, and contextualize spatially resolved molecular pathology data, with a particular emphasis on RNA-based spatial technologies.

The project aims to bridge raw spatial omics data, pathological tissue context, and reproducible computational analysis through a schema-driven design.

Core Design Principles

  • Schema-first design: data structures are defined before data ingestion.
  • Spatial context as a first-class entity: spatial information is not auxiliary metadata.
  • Modality-agnostic, pathology-aware: multiple technologies, consistent interpretation.
  • Reproducibility over volume: clarity and provenance are prioritized.

Data Layers (Schema v0.2)

Layer Description Status
Sample Biological origin, condition, and acquisition metadata Available
Spatial Coordinates, tissue geometry, and regions of interest Partial
Molecular RNA expression and other molecular measurements Planned
Annotation Cell types, niches, and pathological structures Planned
Derived Computed spatial features and representations Planned

Access Model (Planned)

RNA-Pathology will support public datasets, controlled-access datasets, and programmatic access via APIs. User accounts and access control are part of the long-term roadmap.

Project Status

RNA-Pathology is currently in an early public architecture phase, focusing on schema definition and documentation.

RNA-Pathology is an open research initiative.
