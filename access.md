# Access

RNA Pathology supports tiered data access to ensure ethical use,
scientific rigor, and sustainable collaboration.

## Access levels

* **Public**: Exploratory datasets and summaries
* **Controlled**: Academic access upon request
* **Collaborative**: Private or pre-publication projects

The database is under active development.
Early access opportunities will be announced.

For inquiries, please see the [Contact](contact.md) page.
